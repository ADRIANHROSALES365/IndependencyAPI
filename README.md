# IndependencyAPI
Personal API Project that helps you to manage monthly fixed expenses. It lets you know how much your expenses cost and give you tailored information about which type of expense it is and what kind of expenses you expend the most of your salary. 


* Framework: .NET 8 / .NET 10 * ORM: Entity Framework (Code First Approach) * Database: SQL * Architecture: MVC Pattern * Mapping: AutoMapper (Ensuring clean separation between Entities and DTOs).
